# Movie Search System

***Database:***

        * NAME
        * PLOT
        * YEAR
        * DIRECTOR
        * IMAGE
        * RATINGS: [imdb, rotten]

* URL Endpoints
    * User:
        * Login
        * Create Account - Post
    * Movies: 
        * HomePage - /movies - GET
        * Movie/ID - /movie/:id - GET
